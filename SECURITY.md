## Security & Intent

This tool was built for one purpose: extracting and optimizing 
Tailwind CSS from HTML files. Nothing more, nothing less.

**What this tool does:**
- Reads your HTML files from `/docs`
- Extracts Tailwind config blocks
- Builds a purged, optimized CSS file
- Writes to `/assets/tailwind.css`

**What this tool does NOT do:**
- No network requests
- No data collection
- No external dependencies at runtime
- No execution of arbitrary code

The source is fully transparent - read it before you run it.  
That's not a suggestion, that's how responsible development works.
